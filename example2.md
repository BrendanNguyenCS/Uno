**This is version 2 of the game where draw four is supported.**

```json
{
  "countPlayers": 6,
  "countInitialCardsPerPlayer": 8,
  "countDigitCardsPerColor": 3,
  "countSpecialCardsPerColor": 3,
  "countWildCards": 3
}
```
